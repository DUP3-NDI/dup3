# dup3 NIDR

Le site se trouve à cette adresse : mathis-jouson.etu.u-bordeaux.fr/nuit-de-l-info
Nous avons décidé de réaliser 3 défis en plus du défi national qui sont :
- "On veut du gros pixel !" du département informatique de l'IUT du Littoral Côte d'Opale
- "Hidden Snake" 
- "Oh les beaux boutons" de l'entreprise AJIR

Nous avons réalisé un site pédagogique, qui permet de voir que certains appareils peuvent avoir des effets différents selon la manière dont ils sont fabriqués et des composants utilisés.
Le site est aussi plutôt ludique et permet en arrivant au bout (plutôt rapidement) de pouvoir jouer au Snake "caché".
